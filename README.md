- 🎨 Modern and responsive design
- 📄 Projects showcase section
- 🛠️ Skills & technologies section
- 🎓 Education background section
- 📬 Contact section with social links
- ⚡ Lightweight — no JavaScript required
- 🌐 Cross-browser compatible

## 🚀 Quick Start

```bash
git clone https://github.com/sa7716834-ux/Portfolio-Maker.git
cd Portfolio-Maker
```

Then simply open `Portfolio.html` in your browser — no installation needed.

## 🚢 Deployment

### GitHub Pages

1. Go to repo **Settings → Pages**
2. Set source to `main` branch
3. Your portfolio will be live at:

```
https://sa7716834-ux.github.io/Portfolio-Maker/Portfolio.html
```

### Netlify

Drop the project folder into [Netlify Drop](https://app.netlify.com/drop) and get a live URL instantly.

---

## 🎨 Customization

Open `Portfolio.html` in any code editor and update:

- Your **name, title, and bio** in the hero section
- Your **projects** in the projects section
- Your **skills** tags
- Your **contact links** (email, LinkedIn, GitHub)
